How To Make a Navigation Bar

And Link Pages to Eachother

  • To create a navigation bar, you need to first create a nav bar class where you can add the div class for the container fluid. After this you make a ul or un-ordered list.
  • Inside the unordered list, you should add as many items to your list as how many pages are in your website.
  • In each section of the list, attach a hyperlink to each item of the link or href, and link into the file used for the other page. You can name the page on the nav bar whatvever you would like.
  • You can also add inside the orginal <li> class="active" which makes the tab on the navagation bar highlighted.
