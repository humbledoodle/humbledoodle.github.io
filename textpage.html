How To Add Text and Format It

Elements How To Use It How To Write It
Text The text element allows you write a single line piece of text <text>
Paragraphs The paragraph element allows you to write multiple lines of text in a singular paragraph <p>
Bold The bold tag is used inside another text element and makes the text bold <b>
Italics The italic tag is used inside another text element and makes the text italicized <i>
