How To Style Your Website

Ways To Style What It does How To Write It
Text Color Changes the color of the text to any color you would like Write the code inside the first line: <style="color:#FFFFFF;">
Backround Color The paragraph element allows you to write multiple lines of text in a singular paragraph <body style="background-color:blue;">
Font Size Changes the font size of the specified area Write this inside of a <styles> in CSS: h1 { font-size: 16px; }
Font Family Changes the font of the specified area Write this inside of a <styles> in CSS: h1 { font-family: sans-serif; }
